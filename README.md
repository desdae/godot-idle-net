# Idle Net

Minimal Godot `.NET` prototype for an effectively infinite 2D map.

## Requirements

- Godot `4.6.1-stable` `.NET` build
- `.NET SDK 8`

## Run

1. Open this folder in the Godot `.NET` editor.
2. Let Godot restore and build the C# project.
3. Press Play.

## Current behavior

- Grass tiles are generated everywhere by default.
- Decorative berries, trees, and stones are placed deterministically from tile coordinates.
- Move with `WASD` or the arrow keys to keep exploring.
